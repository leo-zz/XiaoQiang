JVM信息包含JVM的CPU、内存、Class、线程等信息

CPU信息

以折线图的形式展示，信息包含可用CPU核心数、JVM的CPU使用率、主机CPU使用率、系统平均负载

内存信息

以折线图的形式展示，信息包含堆、非堆内存的使用情况，主机物理内存和虚拟内存，内存池（上次GC后的内存使用、内存使用、名称、类型、内存峰值、内存管理者）

Class信息

以折线图的形式展示，信息包含目前加载的类的数量，总加载类的数量，卸载的类的总数量，类文件的路径（bootstrap、system）

线程信息

以折线图的形式展示，信息包含线程峰值、启动的线程总数、当前存活的线程数、死锁线程数、对象死锁线程数

GC信息包含GC耗时、内存使用情况等信息

GC信息

以折线图的形式展示，信息包含多个GC收集器信息（GC总次数、总耗时，GC前的内存使用情况、GC后的内存使用情况，GC的耗时和id，开始和结束时间）
